UniRAG — Multimodal Agentic RAG Platform

Students and professionals drown in documents — lecture slides, research papers, reports — with no easy way to ask questions and get intelligent answers. UniRAG solves this by letting you upload any document and have a conversation with your knowledge base.

I built the full backend: a multimodal RAG pipeline that accepts text, images, and PDFs as input, retrieves semantically relevant context, and generates grounded answers using an agentic LangChain orchestration layer. Asynchronous ingestion via Celery and Redis keeps the system fast even under load.

The platform also includes a React frontend for document management and chat — and is designed to be fully extensible for new file formats, retrieval strategies, and models.

Voice Controlled Wheelchair

Millions of people depend on assistive mobility — yet most solutions still require physical effort. This project brings hands-free control to wheelchairs using voice commands, making independence truly accessible.

I designed and implemented the full software pipeline: a voice recognition system running on an ESP32 microcontroller using TensorFlow Lite, enabling real-time on-device inference without cloud dependency.

Co-authored a research paper comparing traditional signal processing methods vs. deep learning approaches for voice recognition in embedded systems.
